Perguntas Frequentes

O Finans é gratuito?

O Finans possui um modelo conceitual com planos simulados para fins educacionais e demonstrativos.

Meus dados estão seguros?

Este projeto demonstra boas práticas de organização e segurança, mas não realiza armazenamento real de dados sensíveis.

Posso usar o Finans em qualquer dispositivo?

Sim. O layout é responsivo e preparado para desktop e dispositivos móveis.
